# JEST 테스트 툴 <-- 사용

  • 페이스북에서 만들었다. (Jasmine 기반)
  • Assertion, Mocking, Spying 기능을 제공

# Mocha

  • Jamine과 다르게 서드파티 Assertion, Mocking, Spying 도구를 사용한다
  • 그만큼 유연하고, 확장성이 뛰어남

# Jasmine

  • 오랜 시간 사용자와 커뮤니티에 의해 생성된 방대한 자료
  • 거의 모든 버전에서 Angular 지원
  • 테스팅 도구가 제공하는 것들
  1. Assertion Function 테스트 대상이 반환하길 기대하는 값을 검사하는 함수이다
  2. Spies 얼마나 많이 호출 되었는지 같은 함수에 대한 정보를 제공한다 특정 함수에 인자를 기록 해두고, 호출을 한 후 값을 반환하거나 예외를 발생 시킨다 알 수 없는 함수의 스파이
  3. Mock and Fake 테스트 하는 과정에서 특정 모듈이나 행동을 조작합니다

# 그 밖에 더 살펴볼 만한 테스팅 도구

  • jsdom
    • 자바스크립트로 WHATWG DOM과 HTML 표준을 구현해서 브라우저 환경을 흉내 낼 수 있다
  • Istanbul
    • 유닛 테스트 커버리지 리포팅 툴
    • 구문, 라인, 브랜치, 함수 단위로 커버리지를 백분율로 제공
  • karma
    • 브라우저 환경에서 테스트를 실행 해주는 실행기 역할
  • Chai
    • BDD/TDD Assertion Library
  • Sinon.js
    • 독립적으로 Test Spy, Stub, Mock 을 제공하는 프레임워크
  • Wallaby
    • IDE에 코드가 수정될 때마다 실시간으로 테스트 하고, 실패시 옆에 표시 해준다. 유료지만 추천하는 사람이 많다고 함
