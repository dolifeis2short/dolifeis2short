# 1. 정보보호의 3요소

  • 기밀성 - 스누핑, 트래픽 분석
  • 무결성 - 변경, 가장, 재연, 부연
  • 가용성 - DoS +) 추적성, 인증, 부인방지

# 2. 암호화 시스템

  • 양방향 (암호화, 복호화)
    • 대칭키
      • 스트림 암호 : 비트 또는 비트 쿢음 단위로 XOR
      • 블록 암호 : 크기를 블록 단위로 나눠 암호화, DES, 3DES, AES
    • 비대칭키 : RSA, DH
  • 일방향(암호화) : MDC(Hash), MAC(Hash+ 대칭키, 전자서명(Hash+비대칭키)
