# HTML

# 1. BLOCK 요소

  • block 요소 ( height, width 적용, 자동 줄바꿈 )
  • (vertical-align , text-align 적용되지 않음)
  div, h1, h2, h3, h4, h5, h6, table, ul, dd, dl, ol, p, pre, hr,
  address, article, aside, audio, blockquote, canvas, fieldset, figcaption, figure, footer, form,
  header, hgroup, noscript, output, section,  video

# 2. inline 요소

  • inline 요소 (자동 줄바꿈 X)
  • (height, width 적용되지 않음)
  button, img, input, label, a, textarea,
  abbr, acronym, b, bdo, big, br, cite, code, dfn, em, i, kbd, map, object, q, samp,
  small, script, select, span, strong, sub, sup, tt, var

# 3. inline-block

  • 줄바꿈이 되지 않는다. width, height 지정 가능. 지정하지 않을 경우, 컨텐츠 만큼 잡힘.

input은 value 값을 주어야 한다. radio는 name을 같이 해서 카테고리를 만든다.

preventDefault() -- 페이지가 이동하지 않게 함. stopPropagation() - 이벤트가 버블링 되지 않게 함.
