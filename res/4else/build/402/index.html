# WEBPACK

# 1. WEBPACK 기본 문법

바벨은 파싱과 출력만 담당하고 변환 작업은 다른 녀석이 처리하데 이것을“플러그인”이라고 부른다.

module.exports = {
    presets: [
        ['@babel/preset-env', {
        targets: {
        chrome: '79', ie:'11'
        }, useBuiltIns: 'usage', // 'entry', false
        corejs: {
        version: 2, //3
    }
    }]
    ]
}

pollyfill을 위해서 core-js 추가

npx babel es6 -w -d es5
(w는 자동감시 옵션, d는 디렉토리 설정)
